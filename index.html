How do I do that?

01 Information Gathering, Planning

I take any idea that comes to mind and start searching on web to get a base cleared if I can do it and if yes then how am I gonna do it.

02 Design, Content Writing

Then I create some wireframes and look for some mock-ups to get the basic design based on the information I gathered.

03 Coding, Testing, Review

Then comes some random thoughts for content and if got data through api's then that. And after that simple testings to see if everthing is working correctly as I wanted.

Then comes hosting the project. I personally prefer firebase and netlify. I also have used github-pages for few of my projects. But by far I find firebase & netlify platforms hosting options the best and easy to use.
